# Crowdflower Test Job

### 运行要求
* MongoDB 服务器
* Nodejs.

###  检出并运行
	mpn install && bower install
    
移动服务器
	npm start 
或者
	nodemon -- preferred
运行地址 
	http://localhost:3000

### Admin 
  管理员可以用默认的 username:admin 和 password:123456 登录, 登录时需要选中 is admin 单选框.
